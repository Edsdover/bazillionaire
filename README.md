# Bazillionaire

Collaborative game development in HTML, CSS and JavaScript.

## Usage

Docker.

```shell
$ docker-compose up
```

###### App

* <http://localhost:3000>

###### API and Explorer

* <http://localhost:3000/api>
* <http://localhost:3000/explorer>

###### OpenAPI Schema

* <http://localhost:3000/explorer/swagger.json>
